This is a simple demo of the JCurl Curling Framework. The Project is in a very early pre-alpha state.

This Demo provides

  • Display of a (hard-coded) rock setup,
  • animation of rock movement,
  • very simple physical model (ice without curl, fully elastic hits neglecting rock rotation)

See the todos for planned improvements.

To view the animation move the mouse to the ice. Use the BLANK key on your keyboard to start and stop.
